Das Document Object Model(DOM) ist eine strukturelle Repräsentation von XML, SVG oder HTML Dokumenten. Mittels JavaScript kann auf das DOM zugegriffen und manipuliert werden. Dies ermöglicht es, weitere Elemente in den DOM einzufügen, zu entfernen, diese anzupassen oder weitere Manipulationen vorzunehmen. Erst dadurch wird es möglich, die statischen HTML Webseiten dynamischer zu gestalten und ständige Reloads zu verhindern DOM. Die DOM Manipulation kann sehr vielseitig eingesetzt werden, wie Sie auch in den weiteren Aufgaben sehen können. Beispielsweise können Eingabefelder bereits Client-seitig überprüfen werden, wodurch zusätzliche Kommunikation mit dem Server eingespart werden kann. An sich können Sie die gesamte Seite viel dynamischer gestalten, indem Sie weitere Elemente zur Laufzeit erzeugen. Um diese Themen geht es in der folgenden Übung.
