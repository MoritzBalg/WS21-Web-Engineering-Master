Web-Komponenten bieten Entwicklern die Möglichkeit eigene HTML-Tags zu definieren. Damit können eigenen Elemente erstellt werden, die Webseitenübergreifend weiderverwendet werden können. Webdesigner können die entworfenen Komponenten deklarativ in Ihrer Webseite bzw. im Frontend einsetzen und müssen sich keine Gedanken über den Code dahinter machen.

8.1.2 - Rednerliste

Mit der Rednerliste können Redner eingetragen werden. Sobald ein neuer Redner angelegt wurde, beginnt ein Timer, der die Redezeit misst. Wird ein weiterer Redner hinzugefügt, stoppt der Timer des ersten Redner und der des zweiten startet. Weiterhin kann der Timer über eine entsprechende Schaltfläche angehalten und fortgesetzt werden.

 a1_2.html

8.1.3 - Tabellenkalkulation

Eine einfache Tabellenkalkulation die es ermöglicht einträge in eine Tabelle vorzunehmen und mit der Formel: "sum=1,1:1,2" Berechnungen durchzuführen. Achtung! die zwei Parameter hinter dem Gleichheitszeichen entsprechen den zellenadressen und müssen ggf. für Ihr Beispiel angepasst werden.
