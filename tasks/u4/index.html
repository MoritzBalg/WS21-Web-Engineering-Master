Mittels JavaScript kann auf das DOM zugegriffen und dies Manipuliert werden. Dies ermöglicht es weitere Elemente in den DOM einzufügen, zu entfernen, diese Anzupassen oder weitere Manipulationen vorzunehmen. Erst dadurch wurde es möglich die statischen HTML Webseiten dynamischer zu gestalten und andauernde Reloads zu verhindern. Die DOM Manipulation kann sehr vielseitig eingesetzt werden, wie Sie auch in den weiteren Aufgaben sehen werden. Beispielsweise könne Sie so Eingabefelder bereits Client-Seitig überprüfen und sparen sich zusätzliche Kommunikkation mit dem Server. An sich können Sie die gesamte Seite viel dynamischer gestalten, indem Sie weitere Elemente zur Laufzeit erst erzeugen. Um diese Themen geht es in der folgenden Übung.

Aufgabe 4.1 - Einkaufsliste

Dom Manipulation

In dieser Aufgabe sollte eine Einkaufsliste gestaltet werden, die neue Einträge in einer Liste erstellt, die in einer Textbox angegeben wurden. Neben dem reinen Text wurden auch Buttons erzeugt, die es erlauben die Einträge aus der Liste wieder zu entfernen.

Einfacher Excel Klon im Browser
