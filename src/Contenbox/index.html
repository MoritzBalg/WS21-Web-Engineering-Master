Content-Box

Die Content Box stellt eine zusammenhängende Einheit dar, die neben dem eigentlichen inhalt und einer Überschrift auch Referenzen beinhalten kann. Der Inhalt dieser Box wird ebenfalls gerendert. So ist es z.B. auch möglich HTML Inhalte in dieser Box anzuzeigen:

Ich bin eine H4

Ich bin eine H5
Eingebunden wird die Content Box über folgende Zeile:

<content-box> headline="Überschrift">Inhalt</content-box>

Dies erzeugt folgende ausgabe: Inhalt
Content Boxen können auch geschachtelt werden Inhalt der äußeren Box Inhalt der inneren Box
