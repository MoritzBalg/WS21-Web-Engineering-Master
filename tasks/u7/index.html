JavaScript hat sich in den vergangenen Jahre immer weiter etabliert. Gerade durch die Google V8-Engine ist auch die Sprache immer performanter geworden und ist inzwischen eine sehr beliebte Programmiersprache. NodeJS adaptiert die Google Engine um JS auch Serverseitig ausführen zu können. Eine Vielzahl an Erweiterungspaketen, die mit dem Node Package Manager (NPM) installiert werden können, bieten einen vergleichsweisen unkomplizierten Einstieg in die Backend Entwicklung. Ein wichtiger Vorteil, den auch Entwickler von Node erkannt haben, ist die Tatsache, dass man mit NodeJS als Backend nur noch eine vorherschende Programmiersprache im Projekt hat.

Aufgabe 7.1 - File Generatoren

Datei-Erzeugung über die Kommandozeile

In der ersten Aufgabe sollten zwei Kommandozeilen-Programme mit NodeJS entworfen werden, die es ermöglichen große Dateien über die Kommandozeile zu erzeugen. Die Datei number_file_gen.js erszellt eine Datei mit x Zeilen und gibt jeweils die Zeilennummer am Anfang aus. Das Programm aplpha_file_gen.js erstellt ebenfalls eine Datei mit x Zeielen.
