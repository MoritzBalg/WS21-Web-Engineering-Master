Vue ist ein Framework für die Client-Seitige JavaScript Entwicklung. Neben der Erstellung von reaktiven Webkomponenten bietet es zahlreiche weitere Features wie Page-Router oder Single Page Applications (SPA) an. Vue zeichnet sich durch einen vergleichsweisen einfachen Einstieg aus und ist vielleicht auch deshalb in den letzten Jahren immer populärer geworden (siehe Abb. 10.1). In dieser Übung werden zwei Webkomponenten mit Vue präsentiert und eine SPA.

Eingabe Auswerter

Sie finden in der Vorschau das Element zwei mal, wodurch gezeigt werden sollte,dass die Komponenten unabhängig voneineander laufen.
