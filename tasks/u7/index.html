JavaScript hat sich in den vergangenen Jahren immer weiter etabliert. Gerade durch die Google V8-Engine ist auch die Sprache immer performanter geworden und ist eine sehr beliebte Programmiersprache. Ryan Dahl hat mit NodeJS die Google-Engine adaptiert, um JS auch serverseitig ausführen zu können. Eine Vielzahl an Erweiterungspaketen, die mit dem Node Package Manager (NPM) installiert werden können, bieten einen vergleichsweisen unkomplizierten Einstieg in die Backendentwicklung. Ein wichtiger Vorteil ist die Tatsache, dass man mit NodeJS als Backend nur noch eine vorherrschende Programmiersprache im Projekt hat.

Aufgabe 7.1 - File Generatoren

Datei-Erzeugung über die Kommandozeile

In der ersten Aufgabe sollten zwei Kommandozeilen-Programme mit NodeJS entworfen werden, die es ermöglichen, große Dateien über die Kommandozeile zu erzeugen. Die Datei number_file_gen.js erstellt eine Datei mit x Zeilen und gibt jeweils die Zeilennummer am Anfang aus. Das Programm aplpha_file_gen.js erstellt ebenfalls eine Datei mit x Zeilen. Im Gegensatz zum ersten Programm wird hierbei nicht die Zeilennummer sondern Buchstaben(Bsp.: A,...,Z,AA) ausgegeben.

 number_file_gen.js alpha_file_gen.js

Dateien zusammenfügen mit Read und Stream
